# Retail Sales Analysis
## Overview
This project showcases the SQL skills and techniques used by data analysts to explore, clean and analyze retail sales data. SQL queries are written to set up a retail sales database, perform exploratory data analysis (EDA) and answer specific business questions.

## Objective
* Set up a database: Given the sales data, create and populate the retail sales database
* Data Cleaning: Remove any missing or null value records
* Exploratory Data Analysis (EDA): Understand the dataset by performing exploratory data analysis
* Business Analysis: Use SQL queries to answer business questions and derive insights from the sales data.
